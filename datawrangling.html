Data Frames

A Data Frame is a list of vectors (columns of data) of equal length.
Example:

# Vectors

name <- c("Kenia", "Sabrina", "Kelvin", "Alex")
age <- c("24", "24", "27", "26")
sex <- c("Female", "Female", "Male", "Male")


# Data Frame

dataframe <- data.frame(name, age, sex)



We can use the print() command to view the dataframe.

     name age    sex
1   Kenia  24 Female
2 Sabrina  24 Female
3  Kelvin  27   Male
4    Alex  26   Male



We can print a single column.

dataframe["name"]
     name
1   Kenia
2 Sabrina
3  Kelvin
4    Alex

We can identify the column by name(above), or index number(below).

dataframe[1]
     name
1   Kenia
2 Sabrina
3  Kelvin
4    Alex



We can print a single row.

dataframe[3, ]
    name age  sex
3 Kelvin  27 Male

Or multiple rows.

dataframe[1:3, ]
     name age    sex
1   Kenia  24 Female
2 Sabrina  24 Female
3  Kelvin  27   Male
# Don't forget the comma!



To load a dataframe from a file on your computer

(You will need the file path for your data file)

mydata <- read.csv("filepath")  #Read a .csv file
mydata <- read.delim("filepath")  #Read other table data formats

Wait: Before we start filtering, you should understand how the Magrittr package allows us to use pipes to tie together multiple dplyr functions into a single executable command.

Below, we can both define the dataframe, and print the table as we did above with one single script of dplyr commands that are piped together with Magrittr “%>%” piping.

starwarsdata<- starwars%>%   #Step 1: Define the Dataframe
  print()  #Step 2: Print the table
